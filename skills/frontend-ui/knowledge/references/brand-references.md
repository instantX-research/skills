## Brand Anti-Patterns

What each brand would NEVER do â€” critical for faithful clones and archetype-appropriate generation.

| Brand | Never do this |
|-------|---------------|
| **Linear** | Purple gradient CTA, stock photography, pastel section backgrounds, rounded-xl cards, centered headline |
| **Stripe** | Dark mode (Stripe stays light), cramped spacing, no-shadow flat cards, Comic Sansâ€“adjacent fonts, chaotic grid |
| **Vercel** | Warm color temperature, decorative illustration, marketing stock photos, serif typography, colorful accents |
| **Anthropic** | Cold blue-white palette, sans-serif for everything, dense information layout, neon accents, startup-style hero |
| **Notion** | High contrast accents, heavy drop shadows, complex animations, decorative icons, branded color splashes |
| **Raycast** | Light mode landing page, no glassmorphism, uniform card sizes (bento requires intentional variation), emoji-heavy copy |
| **Apple** | Cluttered layouts, text-heavy above-the-fold, generic stock photos, gradients on buttons, visible grid lines |
| **A24** | Startup gradient hero, feature comparison tables, sans-serif display headings, CTAs on every section |
