# Curated Reference Sites by Scene

**When to use:** Load this file in Phase 1 when the user's scene type is identified and no stronger visual guidance exists. Use `WebFetch` to inspect 1–2 sites from the matching section. Extract information architecture, layout patterns, and UI conventions — do not copy branding literally.

Each section lists what to look for so fetches are targeted and efficient.
