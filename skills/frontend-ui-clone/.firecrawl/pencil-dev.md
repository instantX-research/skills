### Design as code

Design files live in your repo. Version, branch, and merge with Git just like you do in development.

## The bi-directional MCP vector canvas you’ve been dreaming about.

Pencil doesn’t just provide MCP reading tools, but also full write access + many other handy tools to fully operate the canvas. This is the real magic. You can plug in the whole world of MCPs, bring in data from other sources like databases, APIs, chart data, Playwright/Puppeteer, or plug in other agents easily. You’re in charge!
